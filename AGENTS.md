# Agent Instructions

## Build & Run
- Maven project in `bancoproyect/`. Run all Maven commands from this directory.
- Run application: `mvn clean javafx:run`
- Main class: `society.App`

## Environment
- Java 11, JavaFX 13
- Project is modular. Update `src/main/java/module-info.java` when adding new packages that require JavaFX access.

## Resources
- FXML files: `src/main/resources/society/`
